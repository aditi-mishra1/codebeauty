What you will learn
Create professional web pages of an application using HTML. Develop web pages having textual content using HTML elements like lists, tables, etc. Create a web application with relevant forms to accept and validate user inputs. Create navigational links between web pages of an application using HTML hyperlinks. Build media-rich web pages with embedded video, audio, and image content using HTML media elements. Develop web pages by embedding required web documents using HTML iFrames. Develop a quality HTML code by incorporating coding best practices and security measures.

React JS

ReactJS is a JavaScript library for building User Interfaces, created by Jordan Walke, a software engineer at Facebook, and open-sourced to the world by Facebook and Instagram team. React can be used basically for developing Single Page Application, helps in developing amazing front end applications, and to design faster websites with an aim to provide speed, simplicity, and scalability. React is used by most of the popular companies because of its notable and unique features like Components, Virtual DOM, JSX, Declarative, and unidirectional data flow. React provides flexibility as it allows us to choose any third-party JavaScript libraries/APIs for accomplishing tasks such as Routing and for communicating with the server. React can follow FLUX or Redux architectures for efficiently managing the state of the application
